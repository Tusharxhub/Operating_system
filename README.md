# 🖥️ Operating System Lab – Learn OS by Doing

This repository contains hands-on shell scripts and experiments to help you understand core Operating System concepts through practical exercises. Scripts are organized by day and topic, covering basics, conditionals, loops, number theory, and exam-style problems.

## 📋 Script Summary

### Day 1 – Basics & Simple Scripts
- **Basic commends.txt**: Linux command reference
- **Main.sh**: Even or odd checker
- **Number.sh**: Largest of two numbers
- **Sum.sh**: Sum of two numbers

### Day 2 – Conditionals & Real-World Logic
- **Alphabate.sh**: Vowel or consonant checker
- **average.sh**: Average of three float numbers
- **circle.sh**: Area of a circle
- **Interest.sh**: Simple interest calculator
- **Job.sh**: Job eligibility by age
- **Largest.sh**: Largest among three numbers
- **Voting.sh**: Voting eligibility checker
- **Year.sh**: Leap year checker

### Day 3 – Loops & Number Theory
- **Divisible.sh**: Numbers divisible by 5 and 7 (50–100)
- **Even.sh**: Even numbers divisible by 5 (50–100)
- **Factorial.sh**: Factorial of a number
- **Number.sh**: Print numbers 1 to N
- **Prime.sh**: Prime number checker

### Day 5 – Deadlock Concepts
- **Avoidance.sh**: Demonstrates deadlock avoidance logic
- **Detection.sh**: Demonstrates deadlock detection logic

### Day 6 – Process Management
- **process.sh**: Create parent and child process and print their PIDs
- **synchronisation.sh**: Parent waits for child completion using `wait`
- **orphan.sh**: Demonstrates orphan process behavior
- **zombie.sh**: Demonstrates zombie process behavior
- **child.sh**: Creates multiple child processes and synchronizes with parent

### Exam – Lab Exam Scripts
- **fibonacci.sh**: Fibonacci sequence generator
- **prime.sh**: Prime number checker

## ▶️ How to Run Scripts

1. Make the script executable:
   ```bash
   chmod +x Day\ 1/Main.sh
   ```
2. Run the script:
   ```bash
   bash Day\ 1/Main.sh
   ```
